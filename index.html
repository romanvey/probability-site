Example 1

Student knows 25 questions from 60 questions. What is the probability to pass the exam? Exam considered to be passed if 2/3 answers are correct.

  • Solution
Let`s define answer $ P(E) $, $ E $ - event that student pass exam.
Student knows 25 questions and doesn`t know 35 questions respectively. For pass the exam student need to know 2 or 3 questions from 3.
Firstly, let`s find all possible combinations to pass exam with 2 right questions. It is just $ C_{25}^{2} \cdot C_{35}^{1}$.
Nextly, let`s find all possible combinations to pass exam with 3 right questions. It would be $ C_{25}^{3}$.
Final answer would be all possible combinations to get 2 right questions and 3 right questions divided by all possible combinations to get 3 questions from 60 questions.
All possible combinations to get 3 questions from 60 equal $ C_{60}^{3}$. $$ P(E) = \frac{C_{25}^{2} \cdot C_{35}^{1}+C_{25}^{3}}{C_{60}^{3}} = \frac{12800}{34200} = \frac{640}{1711} \approx 0.37$$

Example 2

In the chess game, k players are played. By rules, everyone has to play with everyone. How many games have been played?

  • Solution
Firstly, let`s consider what we need to find. If you look attentively you mention that answer would be all possible combinations of getting 2 players from k players list. So, answer just: $$ \binom{k}{2} $$

Example 3

Three passengers get on the elevator in 20 storied house. What is the probability that:
a) all of them get off the elevator at different flats?
b) all of them get off the elevator at same flat?
c) two of them get off the elevator at same flat?

  • Solution
a) Let`s define answer $ P(A) $, $ A $ - event where all of them get off the elevator at different flats. $ P(A) = \frac{favorable\ outcomes}{all\ outcomes}$
$all\ outcomes = A_{3}^{19}$, because everybody has 19 possible flats to get off.
$favorable\ outcomes = 19\cdot18\cdot17$, everybody has one less possible flat, because it became occupied, but we need situation were all people have different flats. $$ P(A) = \frac{19\cdot18\cdot17}{19\cdot19\cdot19} = \frac{306}{361} \approx 0.84$$ b) Let`s define answer $ P(B) $, $ B $ - all of them get off the elevator at same flat. $all\ outcomes = A_{3}^{19}$, because everybody has 19 possible flats to get off.
$favorable\ outcomes = 19$, first favorable outcome - when all passengers get off at first flat, second favorable outcome - when all passengers get off at second flat etc. $$ P(A) = \frac{19}{19\cdot19\cdot19} = \frac{1}{361} \approx 0.0027$$ c) Let`s define answer $ P(C) $, $ C $ - event where two of them get off the elevator at same flat. It`s much harder problem if we would solve it in combinatoric way like previous problems. But we can notice $ P(A) + P(B) + P(C) = 1$, so $ P(C) = 1 - \frac{306}{361} -\frac{1}{361} = \frac{54}{361} \approx 0.15$

Example 4

There are 500 students in a high school senior class. Of these 500 students, 300 regularly wear a necklace to school, 200 regularly wear a ring, and 125 regularly wear a necklace and a ring.

Let N be the event that a randomly selected senior regularly wears a necklace and R be the event that a randomly selected senior regularly wears a ring.

Using this information, answer each of the following questions.

  • Solution

What is P(N), the probability that a senior wears a necklace?

300 seniors wear a necklace, 500 total students. Thus => probability is
$ P(N) = \frac{300}{500} = \frac{3}{5} = 0.6 $

What is P(R), the probability that a senior wears a ring?

The same as above; 200 seniors wear ring, 500 total, thus probability is
$ P(R) = \frac{200}{500} = 0.4 $

What is P(N and R), the probability that a senior wears a necklace and a ring?

$ P(N and R) = \frac{125}{500} = 0.25$

What is P(N or R) the probability that a senior wears a necklace or a ring?

*by formula:* $ P(N or R) = P(N) + P(R) - P(N and R) = 0.6 + 0.4 - 0.25 = 0.75$
by bruteforce: Look at the picture above. 175 seniors wear necklace, but not ring, 75 seniors wear rings but not necklace and 125 seniors wear both ring and necklace, total 175 + 75 + 125 = 375 students wearing necklace or ring
$P(N or R) = 375 / 500 = 0.75$

Example 5

In how many ways can 10 undistinguishable apples be distributed among 5 children?

  • Solution
We have n=10 objects (apples) and k=5 classes (children).
To create k classes we need k-1 separators. And there are n+k+1 possible positions for the separators to be placed in (as we can give 0 apples to some kid(s), we can put separators next to each other) Answer: n + k + 1 = 10 + 5 + 1 = 16ways
